Library.Java
ساخته شده که در فایل اول شما امکان ایجاد آبجکت کتاب رو دارید با نام
کتاب و نام نویسنده.
دیتا ها از نوع پرایوت هستند و قابل دسترسی نیستند بصورت مستقیم.

سه Attribute برای کلاس کتاب درنظر گرفته شده 
name,author,status
که برای status همه کلاس های ایجاد شده مقدار اولیه Available وجود دارد که به فرض این هست که کتابی که وارد کتابخانه میشه موجود هست.

داده های کتاب ها و نویسنده های موجود در برنامه بصورت دستی وارد شدند.

در کلاس Library با استفاده از کلاس استرینگ 
اول اسم کتاب رو به حروف کوچیک تبدیل میکنیم و بعد با ورودی یوزر مقایسه میکنیم و ابجکتی که شامل اون رشته باشه رو برمیگردونیم.

و در آخر یه پرامپت ساده از یوزر میگیریم که کتاب رو میخواد از کتابخونه قرض بگیره یا نه که مقدار عضو داده ای status رو با استفاده از تابع changeStatus از available به borrowed تغییر میدیم.
